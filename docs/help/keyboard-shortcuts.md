# Keyboard Shortcuts

## Global Shortcuts

These shortcuts work anywhere in the application.

| Shortcut         | Action                              |
|------------------|-------------------------------------|
| Cmd+S            | Save current pipeline               |
| Cmd+Z            | Undo last action                    |
| Cmd+Shift+Z      | Redo last undone action             |
| Cmd+K            | Open command palette                |
| Cmd+1            | Switch to Projects view             |
| Cmd+2            | Switch to Pipeline Editor view      |
| Cmd+3            | Switch to Blocks view               |
| Cmd+4            | Switch to Datasets view             |
| Cmd+5            | Switch to Runs view                 |
| Cmd+6            | Switch to Results view              |
| Cmd+7            | Switch to Settings view             |

## Pipeline Execution

| Shortcut         | Action                              |
|------------------|-------------------------------------|
| Cmd+Shift+R      | Run / re-run the current pipeline   |
| Shift+R          | Re-run from selected node           |
| Cmd+Shift+M      | Open / toggle the monitor panel     |
| Escape           | Cancel the current run              |

## Pipeline Editor

| Shortcut         | Action                              |
|------------------|-------------------------------------|
| G                | Toggle guide overlay                |
| Cmd+A            | Select all blocks on the canvas     |
| Cmd+Shift+C      | Clone the current pipeline          |
| Delete           | Delete selected block(s)            |
| Backspace        | Delete selected block(s)            |
| Cmd+D            | Duplicate selected block            |
| Cmd+F            | Find block on canvas                |

## Navigation

| Shortcut         | Action                              |
|------------------|-------------------------------------|
| Cmd+K            | Open command palette                |
| Cmd+,            | Open workspace settings             |
| Cmd+P            | Quick switch between pipelines      |

## Text Editing

Standard text editing shortcuts apply in input fields and text areas.

| Shortcut         | Action                              |
|------------------|-------------------------------------|
| Cmd+C            | Copy                                |
| Cmd+V            | Paste                               |
| Cmd+X            | Cut                                 |
| Cmd+A            | Select all                          |

## Notes

- On Windows and Linux, replace **Cmd** with **Ctrl** for all shortcuts.
- Shortcuts may be overridden when a text input field is focused.
- Use the command palette (**Cmd+K**) to discover additional actions not listed here.
